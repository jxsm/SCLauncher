## 📖 项目简介

SCLauncher 是一个专为《生存战争》游戏设计的现代化启动器，提供版本管理、模组安装、一键启动等功能，让游戏管理更加简单便捷。

### ✨ 功能特性

#### 🎮 核心功能
- **一键启动** - 快速启动游戏，支持多版本管理
- **版本管理** - 安装多个游戏版本，轻松切换
- **版本下载** - 自动从清单文件下载游戏版本，支持断点续传
- **模组管理** - 简单的模组安装/卸载功能，支持启用/禁用
- **进度显示** - 实时显示下载和安装进度
- **主要版本** - 设置默认启动版本

#### 🎨 界面设计
- **简洁现代** - 扁平化设计，清爽的视觉体验
- **响应式布局** - 适配不同分辨率
- **流畅动画** - 细腻的交互体验
- **暗色主题** - 护眼的暗色界面

#### 🔧 技术特性
- **轻量级** - 体积小、启动快
- **高性能** - 原生性能，流畅体验
- **数据持久化** - 本地数据库，安全可靠

## 🚀 使用说明

### 安装

1. 下载最新版本的 SCLauncher 的exe文件
2. 首次运行会自动创建必要的目录结构

### 基本使用

#### 首次使用

1. **配置清单文件**
   - 打开"设置"页面
   - 设置版本清单文件的 URL
   - 清单文件包含所有可下载的游戏版本信息

2. **下载游戏版本**
   - 进入"版本下载"页面
   - 选择"插件版"或"联机版"
   - 点击版本卡片上的"下载"按钮
   - 输入自定义版本名称
   - 等待下载完成

3. **启动游戏**
   - 在"首页"或"已安装版本"页面
   - 点击"启动"按钮即可开始游戏

#### 版本管理

- **查看已安装版本**
  - 进入"已安装版本"页面
  - 查看所有已安装的游戏版本

- **设置主要版本**
  - 点击版本卡片上的"设为主要"按钮
  - 主要版本会在首页显示，且优先启动

- **重命名版本**
  - 点击"改名"按钮
  - 输入新的版本名称

- **删除版本**
  - 点击"删除"按钮
  - 确认后即可删除版本及其所有文件

- **打开版本文件夹**
  - 点击"打开文件夹"按钮
  - 可以直接访问游戏安装目录

#### 模组管理

1. **选择版本**
   - 进入"模组管理"页面
   - 从下拉菜单中选择要管理的版本

2. **导入模组**
   - 点击"导入模组"按钮
   - 选择模组文件（.zip 或 .scmod）
   - 模组会自动安装到选中版本的 mods 目录

3. **启用/禁用模组**
   - 勾选模组前的复选框以启用
   - 取消勾选以禁用模组
   - 禁用的模组会被重命名为 .disable 后缀

4. **删除模组**
   - 点击模组卡片上的"删除"按钮
   - 确认后即可删除模组文件

5. **打开模组文件夹**
   - 点击"打开文件夹"按钮
   - 直接访问该版本的 mods 目录

#### 下载页面

- **分组显示**
  - 版本按游戏版本号分组显示（如 2.31、2.4、2.3）
  - 点击分组标题可展开/收起该组

- **筛选版本**
  - 使用下拉菜单选择"插件版"或"联机版"
  - 只显示对应类型的版本

- **下载进度**
  - 下载时显示实时进度条
  - 可切换到其他页面，进度不会丢失

### 设置说明

- **清单文件 URL**：版本清单文件的地址，包含所有可下载的版本信息
- **最大并发下载数**：同时进行的下载任务数量上限
- **游戏路径**：游戏安装的基础目录

## 📁 目录结构

启动器会在以下位置创建目录：

```
SCLauncher/
├── versions/             # 游戏版本目录
│   └── [版本ID]/        # 各个版本的独立目录
│       ├── game.exe     # 游戏可执行文件
│       ├── data/        # 游戏数据
│       └── mods/        # 模组目录
└── downloads/            # 临时下载目录
```

## ⚠️ 注意事项

1. **首次使用**：需要先配置清单文件 URL 才能下载游戏版本
2. **网络连接**：下载游戏版本需要稳定的网络连接
3. **磁盘空间**：确保有足够的磁盘空间存储游戏文件
4. **版本隔离**：每个版本独立安装，互不影响
5. **模组兼容**：请注意模组与游戏版本的兼容性

## 🐛 常见问题

**Q: 下载失败怎么办？**
A: 检查网络连接和清单文件 URL 是否正确，可以尝试重新下载。

**Q: 游戏无法启动？**
A: 确认游戏文件完整，可以在已安装页面打开文件夹检查文件是否存在。也有可能提供的游戏版本文件不完整，可以反馈一下

**Q: 模组不生效？**
A: 确认模组已启用（复选框勾选），且模组与游戏版本兼容。

**Q: 如何切换游戏版本？**
A: 在首页或已安装版本页面选择要启动的版本，点击"启动"按钮即可。
